# node-api

This is an example of a simple REST api used for educational purposes

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

It needs to have installed:
- npm 6.13.4 (package manager for the JavaScript programming language)
https://www.npmjs.com/get-npm

- Node version: v12.14.1
https://nodejs.org/en/download/package-manager/

### Installing

With the project management tool the dependencies will be downloaded into the node_modules folder from the npm repository using the command:
```bash
npm install
```

### Run

Use the command:
```bash
npm run start
```
